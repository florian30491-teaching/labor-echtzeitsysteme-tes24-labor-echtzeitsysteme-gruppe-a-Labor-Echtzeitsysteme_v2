# Labor Echtzeitsysteme – DHBW Stuttgart

Willkommen im Repository für das **Labor Echtzeitsysteme** an der DHBW Stuttgart.

Dieses Repository enthält die Firmware-Basis sowie die vollständige Aufgabendokumentation des Labors. Die Bearbeitung und Abgabe jeder Aufgabe erfolgt über einen **Pull Request** in diesem Repository.

---

## 📁 Struktur

```
Labor-Echtzeitsysteme/
├── Firmware/          # STM32-Firmware (Basis-Projekt, nicht verändern)
├── Dokumentation/     # Aufgabendokumentation der Studierenden
│   ├── Aufgabe_01/
│   ├── ...
│   └── Aufgabe_08/
├── Bewertung/         # 🔒 Nur für Dozierende
├── Laborhandbuch.md     # Zentrales Nachschlagewerk für Studierende
└── README.md          # Diese Datei
```

## 🚀 Wie arbeite ich mit diesem Repository?

Den vollständigen Workflow (Branch erstellen, dokumentieren, Pull Request stellen) findest du im **[Laborhandbuch](./Laborhandbuch.md)**.

---

## ℹ️ Hinweise

- Die Firmware in `Firmware/` dient als gemeinsame Basis und wird vom Dozenten gepflegt.
- Bilder, Schaltpläne und sonstige Materialien werden **flach** direkt im jeweiligen Aufgabenordner abgelegt.
- Der Ordner `Bewertung/` ist ausschließlich für den Dozenten vorgesehen und wird von Studierenden **nicht bearbeitet**.
