# STM32-Dokumentation

Dieses Verzeichnis enthält Referenzdokumente und Datenblätter für das im Labor verwendete STM32H563-Nucleo-Board.

## Dateien

| Datei | Beschreibung |
|-------|-------------|
| `pinout_nucleo_board.png` | Pinout-Übersicht des Nucleo-144-Boards |
| `pm0264-…-programming-manual-stmicroelectronics.pdf` | Programming Manual – STM32 Cortex-M33 |
| `rm0481-…-stmicroelectronics.pdf` | Reference Manual – STM32H562/H563/H573 |
| `stm32h562ag.pdf` | Datenblatt – STM32H562AG |
| `um3115-…-stmicroelectronics.pdf` | User Manual – STM32H5 Nucleo-144 Board (MB1404) |

## Hinweis zu Git LFS

> **Auf Git LFS wird in diesem Repository bewusst verzichtet.**
>
> Die PDF- und Bilddateien werden direkt als reguläre Git-Objekte gespeichert.
> Dies reduziert die Komplexität für Studierende beim Klonen und Arbeiten mit
> dem Repository erheblich, da kein zusätzliches LFS-Setup erforderlich ist.
>
> Die größte Datei (Reference Manual) ist ca. 61 MB groß und liegt damit
> unterhalb des GitHub-Limits von 100 MB pro Datei.
